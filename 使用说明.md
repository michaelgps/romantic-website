# 💝 生日网站使用说明

一个星空主题、手机端完美适配的生日礼物网站，重点是互动小彩蛋。

## 📁 文件结构
```
romantic website/
├── index.html      ← 网页本体（一般不用改）
├── style.css       ← 样式（想换颜色可以改）
├── script.js       ← 所有内容都在这里改！⭐
├── photos/         ← 把照片放这里
└── music/          ← 把背景音乐放这里
```

## ✏️ 第一步：填内容（只改 script.js 顶部）

打开 `script.js`，最上面的 `CONFIG` 区域就是你要改的全部：

1. **`name`** — 改成她的名字（如 `"小美"`）
2. **`starMessages`** — 互动星座里每颗星星点亮后显示的话，建议 5~7 句
3. **`photos`** — 把照片放进 `photos/` 文件夹，然后把文件名填进去：
   ```js
   photos: ["1.jpg", "2.jpg", "3.jpg"],
   ```
4. **`letter`** — 你想对她说的情书内容（会有打字机效果）

## 🎵 第二步：加背景音乐（可选）

把你们的歌（mp3 格式）放进 `music/` 文件夹，命名为 `song.mp3` 即可。
> 注意：手机浏览器通常需要用户点一下才会播放，所以网站里有个右上角的 ♪ 按钮可以手动开关。

## 👀 第三步：本地预览

直接双击 `index.html` 用浏览器打开就能看效果。
（建议用 Chrome / Edge，并按 F12 → 点手机图标，预览手机端效果）

## 🚀 第四步：发布成链接（发给她）

最简单的方式（免费、3 分钟搞定）：

### 方法 A：Netlify Drop（最简单，零账号也行）
1. 打开 https://app.netlify.com/drop
2. 把整个 `romantic website` 文件夹拖进去
3. 等几秒，它会给你一个链接，发给她就行 ✨

### 方法 B：Vercel
1. 注册 https://vercel.com
2. 上传文件夹或连接 GitHub，自动部署

## 🎨 想改颜色？
打开 `style.css` 最上面的 `:root`，改那几个颜色值即可：
- `--accent` 主粉色
- `--accent-2` 蓝紫色
- `--gold` 金色
